# 教材ファイル リネーム作業マニュアル

対象：アルバイトスタッフ・指導員

---

## 1. 命名規則の概要

ファイル名は以下の順番で各要素を `_`（アンダースコア）でつなぎます。

```
[前置き番号]_[学年]_[クラス]_[科目]_[前後期]_[第XX回]_[教材種別]_[備考]
```

### 具体例

```
01_5年_S_算数_前期_第01回_宿題プリント_約数と倍数の利用
01_5年_Z_理科_前期_第01回_確認テスト_問題_季節と生物
02_5年_Z_社会_前期_第02回_確認テスト_問題解答_くらしに役立つ資源
```

---

## 2. 各項目の詳細

| 項目 | 選択肢 | 補足 |
|---|---|---|
| 前置き番号 | 01, 02, 03… | フォルダ内の並び順を整えるための番号 |
| 学年 | 3年, 4年, 5年, 6年 | |
| クラス | S, A, B, C, Z | 全クラス共通のファイルは「Z」 |
| 科目 | 算数, 数学, 国語, 理科, 社会, 理社, 英語 | 理社は4年のみ使用 |
| 前後期 | 前期, 後期 | |
| 回数 | 第01回, 第02回… | 必ず2桁で記入 |
| 教材種別 | 宿題プリント / 確認テスト_問題 / 確認テスト_解答 / 確認テスト_問題解答 / 体験用教材 / 授業プリント / 入試演習 | |
| 備考 | 例：約数と倍数の利用 | 省略可 |

### 基本ルール

- 区切りは必ず `_`（アンダースコア）を使う
- 数字はすべて半角・2桁ゼロ埋め（例：01, 02）
- スペースは使わない

---

## 3. ファイル名ジェネレーターの使い方

ファイル名は手入力せず、専用ツールで生成してください。

🔗 [ファイル名ジェネレーターを開く](https://camelllllllllll.github.io/file-name-generator/)

### 手順

1. URLをブラウザで開く
2. 各項目をプルダウンで選択・入力する
3. 画面下部に生成されたファイル名を確認する
4. 「コピー」ボタンをクリック
5. Dropboxのファイル名にペーストする（Ctrl+V）

---

## 4. Dropboxでのリネーム手順

1. Dropboxをブラウザまたはアプリで開く
2. 対象ファイルを右クリック →「名前を変更」を選ぶ
3. 既存のファイル名をすべて削除する
4. ジェネレーターでコピーしたファイル名を貼り付ける（Ctrl+V）
5. Enterキーで確定する
6. 進捗管理スプレッドシートの該当行に「完了」を記入する

---

## 5. よくある間違い・NG例

| NG例 | 何が問題か |
|---|---|
| `5年S算数前期第01回宿題プリント` | `_` で区切られていない |
| `01_５年_S_算数_前期_第０１回_…` | 数字が全角になっている |
| `01_5年_S_算数_前期_第1回_…` | 回数が1桁（「第01回」が正しい） |
| `01_5年_S_算数_前期_第01回_宿題プリント ` | 末尾にスペースが入っている |

## 6. 過去問の命名規則

過去問は通常の教材と構造が異なるため、別ルールで管理します。
ジェネレーターは使わず、以下のルールに従って手入力してください。

### 学校配布の過去問

```
[学校名]_[年度]_[試験種別]_[科目]_[種別]
```

| 項目 | 内容 | 例 |
|---|---|---|
| 学校名 | 正式な学校名（省略しない） | 芝, 開智・開智所沢 |
| 年度 | 西暦4桁 | 2026 |
| 試験種別 | 月（数字）または試験名 | 2月, 特待B |
| 科目 | フル表記 | 国語, 算数 |
| 種別 | フル表記 | 問題, 解答用紙, 問題・解答 |

**具体例**

```
芝_2026_2月_国語_解答用紙
開智・開智所沢_2026_特待B_国語_解答用紙
```

---

### 市販の過去問集

```
[出版社]_[シリーズ名]_[科目]_[年度]_[種別]_[対象クラス]
```

| 項目 | 内容 | 例 |
|---|---|---|
| 出版社 | フル表記 | みくに出版, 声の教育社 |
| シリーズ名 | フル表記 | 銀本, 有名中学 |
| 科目 | フル表記 | 国語, 算数 |
| 年度 | 西暦4桁 | 2008 |
| 種別 | フル表記 | 解答, 解答用紙, 問題 |
| 対象クラス | S / A / B / C / Z（省略可） | Z |

**具体例**

```
みくに出版_銀本_国語_2008_解答_Z
声の教育社_有名中学_国語_2005_解答用紙
```

---

## 7. 困ったときは

- ファイル名に迷ったら **ジェネレーターを使う**
- 前置き番号が分からない場合は **指導員に確認する**
- 作業対象のファイルは **進捗管理スプレッドシート** で確認する
